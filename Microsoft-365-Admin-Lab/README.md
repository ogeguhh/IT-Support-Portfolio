# Microsoft 365 Administration Lab

## Overview
This project demonstrates basic administration tasks using Microsoft 365 tools commonly used in IT support environments.

---

## Tasks Performed

### User Account Support
- Assisted with password reset scenarios
- Verified login access to accounts

### Outlook Support
- Troubleshot email login issues
- Verified account connectivity
- Assisted with sending and receiving emails

### Microsoft Teams
- Verified user access to Teams
- Assisted with joining meetings
- Troubleshot connection issues

### OneDrive
- Verified file access and syncing
- Assisted with uploading and downloading files
- Checked storage availability

---

## Skills Practiced
- Microsoft 365 troubleshooting
- User account support
- Application-level issue resolution
- Supporting business productivity tools

---

## Real-World Relevance
Microsoft 365 is widely used in business environments. This lab reflects common IT support responsibilities such as helping users access email, join meetings, and manage files.
