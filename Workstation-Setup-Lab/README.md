# Workstation Setup & Network Configuration Lab

## Overview
This project demonstrates setting up a workstation and configuring basic network connectivity in an IT support environment.

## What I Did
- Set up a computer workstation from initial startup
- Connected the system to a Wi-Fi network
- Verified internet connectivity
- Configured basic system settings
- Ensured the system was ready for software installation

## Troubleshooting Performed
- Diagnosed connection issues when Wi-Fi was not detected
- Restarted network services and reconnected to network
- Verified IP address and connectivity

## Tools Used
- Windows OS
- Network settings (Wi-Fi configuration)

## Skills Practiced
- Workstation setup
- Network connectivity troubleshooting
- System readiness and verification

## Real-World Relevance
These tasks reflect common responsibilities of an IT support technician when preparing systems for users in a business environment.
