# Troubleshooting Scenarios

## Overview
This project demonstrates common IT support troubleshooting situations and how they are resolved.

---

## Scenario 1: User Cannot Log In

### Problem
User reports they are unable to log into their computer.

### Steps Taken
- Checked username and password
- Verified caps lock was not enabled
- Reset the user’s password
- Checked if the account was locked

### Result
User was able to log in successfully after password reset.

---

## Scenario 2: Printer Not Working

### Problem
User reports the printer is not printing documents.

### Steps Taken
- Checked if the printer was powered on
- Verified network connection
- Restarted the printer
- Cleared print queue
- Reinstalled printer driver

### Result
Printer resumed normal operation.

---

## Scenario 3: No Internet Connection

### Problem
User cannot access the internet.

### Steps Taken
- Checked Wi-Fi connection
- Restarted router
- Ran network troubleshooting tool
- Verified IP address

### Result
Internet connection restored.
