# Active Directory Home Lab

## Overview
This project simulates a real-world IT support environment using Active Directory.

## What I Did
- Created user accounts in Active Directory
- Reset passwords and unlocked accounts
- Added users to security groups
- Tested login scenarios

## Tools Used
- Windows Server (Virtual Machine)
- Active Directory Users and Computers

## Skills Practiced
- User account management
- Troubleshooting login issues
- Understanding permissions and access control

## Real-World Relevance
This lab reflects common IT support tasks such as helping users log in, resetting passwords, and managing access in a business environment.
