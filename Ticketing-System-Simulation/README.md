# Ticketing System Simulation

## Overview
This project simulates how IT support tickets are handled in a real-world environment.

---

## Ticket 1: User Cannot Access Email

### Issue
User reports they cannot access their email account.

### Troubleshooting Steps
- Verified username and password
- Checked internet connection
- Reset password
- Confirmed login through Outlook

### Resolution
User successfully logged into their email after password reset.

---

## Ticket 2: Computer Running Slow

### Issue
User reports system performance is very slow.

### Troubleshooting Steps
- Checked running applications
- Closed unnecessary programs
- Restarted the computer
- Checked for updates

### Resolution
System performance improved after restart and closing background apps.

---

## Ticket 3: Printer Offline

### Issue
User reports printer is showing offline.

### Troubleshooting Steps
- Verified printer power and connection
- Restarted printer
- Cleared print queue
- Reinstalled printer driver

### Resolution
Printer returned to online status and resumed printing.
