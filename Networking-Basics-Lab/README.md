# Networking Basics Lab

## Overview
This project demonstrates basic networking concepts and troubleshooting techniques used in IT support.

---

## Tasks Performed

### Checking IP Configuration
- Used command prompt to run ipconfig
- Identified IP address, subnet mask, and default gateway

### Testing Connectivity
- Used ping command to test connection to websites
- Verified network communication

### DNS Verification
- Tested domain name resolution
- Confirmed that websites could be reached by name

### Troubleshooting Network Issues
- Restarted network adapter
- Reconnected to Wi-Fi network
- Verified internet access after reconnecting

---

## Skills Practiced
- Basic networking concepts
- Connectivity troubleshooting
- Command-line tools (ipconfig, ping)
- Understanding how devices connect to networks

---

## Real-World Relevance
These tasks reflect common IT support responsibilities such as diagnosing internet issues, verifying network connectivity, and resolving user connection problems.
